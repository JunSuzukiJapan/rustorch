Model Quantization for compression and acceleration モデル量子化による圧縮と高速化

This module provides comprehensive quantization techniques for neural networks, including post-training quantization, quantization-aware training, and various quantization schemes (int8, int4, float16, etc.). ニューラルネットワーク用の包括的な量子化技術を提供、 ポストトレーニング量子化、量子化対応訓練、様々な量子化スキーム（int8、int4、float16など）を含む。

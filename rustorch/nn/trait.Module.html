ニューラルネットワークモジュールのトレイト A trait for neural network modules.

Required Methods§

Source

fn forward(&self, input: &Variable<T>) -> Variable<T>

モジュールの順伝搬を実行します。 Performs the forward pass of the module.

Source

fn parameters(&self) -> Vec<Variable<T>>

モジュールのパラメータへの参照を返します。 Returns a reference to the module’s parameters.

Source

fn as_any(&self) -> &dyn Any

ダウンキャストのための&dyn Anyとしてのモジュールへの参照を返します。 Returns a reference to the module as &dyn Any for downcasting.

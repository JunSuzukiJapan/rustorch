Computer vision module providing image transforms, data augmentation, and built-in datasets 画像変換、データ拡張、組み込みデータセットを提供するコンピュータビジョンモジュール Computer Vision utilities for RusTorch RusTorch用コンピュータビジョンユーティリティ

This module provides computer vision functionality similar to torchvision, including image transformations, data augmentation, and built-in datasets.

このモジュールはtorchvisionと同様のコンピュータビジョン機能を提供し、 画像変換、データ拡張、組み込みデータセットを含みます。

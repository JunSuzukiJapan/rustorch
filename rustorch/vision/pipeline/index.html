Data transformation pipeline for computer vision コンピュータビジョン用データ変換パイプライン

This module provides advanced pipeline functionality for chaining multiple transformations with conditional logic, parallel processing, and caching.

このモジュールは、条件付きロジック、並列処理、キャッシュ機能を持つ 複数変換のチェーン化のための高度なパイプライン機能を提供します。

Image transformations for computer vision コンピュータビジョン用画像変換

This module provides image transformation functions similar to torchvision.transforms, including basic transformations, data augmentation, and composition utilities.

このモジュールはtorchvision.transformsと同様の画像変換機能を提供し、 基本変換、データ拡張、合成ユーティリティを含みます。

Convenient macro for creating tensors with literal syntax

This macro supports 1D, 2D, and 3D tensor creation with automatic shape inference. All numeric types that implement ToPrimitive are supported and will be converted to f32.

ยงExamples

use rustorch::tensor;

// 1D tensor: shape [4]
let t1 = tensor!([1, 2, 3, 4]);

// 2D tensor: shape [2, 3]
let t2 = tensor!([
    [1, 2, 3],
    [4, 5, 6]
]);

// 3D tensor: shape [2, 2, 2]
let t3 = tensor!([
    [[1, 2], [3, 4]],
    [[5, 6], [7, 8]]
]);

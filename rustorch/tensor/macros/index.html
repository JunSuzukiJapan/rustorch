§Tensor Creation Macros

テンソル作成マクロ

This module provides convenient macros for creating tensors with literal syntax. Supports 1D, 2D, and 3D tensor initialization with automatic shape inference.

§Examples

use rustorch::tensor;

// 1D tensor
let t1 = tensor!([1, 2, 3, 4]);

// 2D tensor
let t2 = tensor!([
    [1, 2, 3],
    [4, 5, 6]
]);

// 3D tensor
let t3 = tensor!([
    [[1, 2], [3, 4]],
    [[5, 6], [7, 8]]
]);

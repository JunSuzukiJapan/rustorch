66//!
67//! **A production-ready deep learning library in Rust with PyTorch-like API, data validation, debugging tools, and enterprise-grade reliability**
68//!
69//! RusTorch v0.5.14 is a fully functional deep learning library that leverages Rust's safety and performance,
70//! providing comprehensive tensor operations, automatic differentiation, neural network layers,
71//! transformer architectures, GPU acceleration, unified error handling system, advanced memory optimization features,
72//! data validation & quality assurance, and comprehensive debug & logging systems.
73//!
74//! ## âœ¨ Key Features
75//!
76//! - **ğŸ”¥ Comprehensive Tensor Operations**: Math operations, broadcasting, indexing, and statistics
77//! - **ğŸ¤– Transformer Architecture**: Complete transformer implementation with multi-head attention
78//! - **âš¡ SIMD Optimizations**: AVX2/SSE4.1 vectorized operations for high performance
79//! - **ğŸ”„ Unified Parallel Operations**: Trait-based parallel tensor operations with intelligent scheduling
80//! - **ğŸ® GPU Integration**: CUDA/Metal/OpenCL support with automatic device selection
81//! - **ğŸ’¾ Advanced Memory Management**: Zero-copy operations, SIMD-aligned allocation, and memory pools
82//! - **ğŸ§  Automatic Differentiation**: Tape-based computational graph for gradient computation
83//! - **ğŸ—ï¸ Neural Network Layers**: Linear, Conv1d/2d/3d, ConvTranspose, RNN/LSTM/GRU, BatchNorm, Dropout, and more
84//! - **ğŸ›¡ï¸ Unified Error Handling**: Single `RusTorchError` type with 61+ specialized helper functions and `RusTorchResult<T>` for cleaner APIs
85//! - **ğŸ”§ Safe Operations**: Type-safe tensor operations with comprehensive error handling and ReLU activation
86//! - **âš™ï¸ Shared Base Traits**: Reusable convolution and pooling base implementations for code efficiency
87//! - **ğŸŒ WebAssembly Support**: Browser-compatible WASM bindings with optimized performance
88//! - **ğŸ” Data Validation & Quality Assurance**: Statistical analysis, anomaly detection, consistency checking, real-time monitoring
89//! - **ğŸ› Comprehensive Debug & Logging**: Structured logging, performance profiling, memory tracking, automated alerts

130//! ## ğŸ—ï¸ Architecture Overview
131//!
132//! The library is organized into several key modules:
133//!
134//! - [`tensor`]: Core tensor operations with parallel and GPU acceleration
135//! - [`nn`]: Neural network layers and building blocks
136//!   - [`nn::safe_ops`]: Safe tensor operations with error handling and ReLU activation
137//!   - [`nn::conv_base`]: Shared base traits for convolution and pooling layers
138//! - [`autograd`]: Automatic differentiation system
139//! - [`vision`]: Computer vision utilities including transforms and datasets
140//! - [`optim`]: Optimization algorithms (SGD, Adam, etc.)
141//! - [`gpu`]: GPU acceleration support (CUDA, Metal, OpenCL)
142//! - [`simd`]: SIMD vectorized operations
143//! - `wasm`: WebAssembly bindings for browser deployment
144//! - [`memory`]: Advanced memory management and pooling
145//! - [`data`]: Phase 5 data loading API with modern `Dataset` and `DataLoader` traits
146//!

149//! RusTorch provides a unified trait-based system for parallel tensor operations:
150//!
151//! ```rust
152//! use rustorch::tensor::{Tensor, parallel_traits::*};
153//!
154//! let tensor1 = Tensor::<f32>::ones(&[4, 4]);  // 2D matrices for simplicity
155//! let tensor2 = Tensor::<f32>::ones(&[4, 4]);
156//!
157//! // Basic tensor operations
158//! let result = &tensor1 + &tensor2; // Element-wise addition
159//!
160//! // Matrix multiplication
161//! let matmul_result = tensor1.matmul(&tensor2);
162//!
163//! // Basic reduction operations
164//! let sum = tensor1.sum();
165//! # assert_eq!(result.shape(), &[4, 4]);
166//! # assert_eq!(matmul_result.unwrap().shape(), &[4, 4]);
167//! ```
